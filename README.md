# Marketing-Analytics-Dashboard
## Project Objective
ShopEasy, an online retail business, is facing reduced customer engagement and conversion rates despite launching several new online marketing campaigns. They want to conduct a detailed analysis and identify areas for improvement in their marketing strategies.
## Data used
- <a href="https://github.com/Phuongthao228/Marketing-Analytics-Dashboard/blob/main/PortfolioProject_MarketingAnalytics.bak">Dataset</a>
## Questions (KPIs)
- Conversion Rate: Percentage of website visitors who make a purchase.
- Customer Engagement Rate: Level of interaction with marketing content (clicks, likes, comments).
- Average Order Value (AOV): Average amount spent by a customer per transaction.
- Customer Feedback Score: Average rating from customer reviews.
## Process
This project follows a structured workflow using SQL, Python, and Power BI. Data was extracted and transformed using SQL, enhanced with sentiment analysis in Python, and visualized through an interactive Power BI dashboard.
